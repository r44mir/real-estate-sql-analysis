# Анализ рынка недвижимости

В проекте использованы данные объявлений о продаже квартир.  
Я посмотрел, как ведёт себя рынок, сколько публикуется объявлений,  
сколько времени квартиры находятся в продаже и как отличается цена  
в разных населённых пунктах.

---

## Инструменты

- SQL (PostgreSQL)
- Yandex Datalens

---

## Что сделано

Сначала данные были обработаны в SQL:

- удалены выбросы по площади, количеству комнат и другим параметрам;
- рассчитана средняя цена за квадратный метр;
- посчитано среднее время продажи квартир;
- сделано сравнение населённых пунктов по основным показателям.

Также посмотрел динамику публикации объявлений по месяцам  
и долю объявлений, которые были сняты с публикации.

---

## Дашборд

По подготовленным данным собран дашборд в **Datalens**.

В нём можно посмотреть:

- количество объявлений;
- среднюю площадь и число комнат;
- изменение средней цены квартир;
- среднее время продажи;
- населённые пункты с наибольшим количеством объявлений;
- сезонность публикации объявлений.

## Выводы

- Основной объём объявлений сосредоточен в Санкт-Петербурге, остальные населённые пункты значительно уступают по количеству предложений
   
- Средняя стоимость квадратного метра существенно различается по регионам, при этом более дорогие объекты сосредоточены в отдельных пригородах
  
- Наблюдается снижение среднего времени продажи квартир, что может указывать на рост спроса или более точное ценообразование  

- На рынке преобладают квартиры с небольшим количеством комнат и средней площадью
  
- Объекты с дополнительными характеристиками (например, наличие парков или водоёмов поблизости) в среднем имеют более высокую стоимость  

- Пик публикации объявлений приходится на середину недели, что может быть связано с активностью пользователей
  
- Выражена сезонность: количество объявлений и активность рынка меняются в течение года  

- В отдельных населённых пунктах квартиры продаются значительно дольше, что может свидетельствовать о низком спросе или завышенных ценах  

## Практическое применение

- Оценка адекватной стоимости объектов недвижимости  
- Определение оптимального времени для размещения объявлений  
- Выявление наиболее ликвидных сегментов рынка  
- Анализ спроса в разных населённых пунктах
